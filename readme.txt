Hide specific price variations from customers in Easy Digital Downloads. Perfect for creating private or archiving pricing options.

== Description ==

EDD Hide Price Variation extends Easy Digital Downloads by allowing store owners to selectively hide specific price variations from customers on the frontend. This is particularly useful when you want to:

* Create private pricing options for specific customers
* Hide certain price variations temporarily
* Manage limited-access pricing tiers
* Control which price options are visible to the general public
* Archive pricing options

= Features =

* Simple checkbox interface to hide/show price variations
* Works seamlessly with existing EDD price variations
* No configuration needed - works out of the box
* Lightweight and efficient

= Requirements =

* WordPress 5.0 or higher
* PHP 7.2 or higher
* Easy Digital Downloads 2.9 or higher

== Installation ==

1. Upload the 'edd-hide-price-variation' folder to the `/wp-content/plugins/edd-hide-price-variation` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit any download product with variable pricing
4. Look for the "Hide this price option from customers" checkbox under each price variation
5. Check the box for any price variation you want to hide from customers

== Frequently Asked Questions ==

= Does this plugin work with the latest version of Easy Digital Downloads? =

Yes, this plugin is regularly tested with the latest version of Easy Digital Downloads.

= Will hidden variations still be available for purchase? =

No, hidden variations will be completely removed from the frontend and cannot be purchased.

= Can I unhide a variation after hiding it? =

Yes, simply uncheck the "Hide this price option from customers" box and update the product.
